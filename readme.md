1️⃣ What is the difference between var, let, and const?
Ans:var: It is an old method. It has function scope, and the same variable can be redeclared and its value can  changed

let:  It has block scope, and the variable cannot be redeclared, but its value can be changed.

const: It has block scope, and the variable cannot be redeclared or changed.
